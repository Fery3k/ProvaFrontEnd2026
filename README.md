# 🚀 Desafio Front-End: Replicação de Interface

Bem-vindo ao seu teste técnico de Front-End! O objetivo deste desafio é avaliar suas habilidades em estruturação, estilização e comportamento de interfaces web.

## 📋 O Desafio

Sua missão é replicar o design e as funcionalidades da interface que será apresentada em tela. O resultado final deve ser o mais fiel possível ao modelo original, tanto visualmente quanto em comportamento.

### Requisitos Técnicos
Para este projeto, você deve utilizar exclusivamente as tecnologias base da web:
- **HTML5**: Semântico e bem estruturado.
- **CSS3**: Layout responsivo (Flexbox/Grid) e estilização.
- **JavaScript (Vanilla)**: Manipulação de DOM e interatividades necessárias.

*Nota: Não é permitido o uso de frameworks (React, Vue, Angular) ou bibliotecas externas (Bootstrap, Tailwind, jQuery), a menos que seja explicitamente autorizado.*

---

## 🛠️ Instruções de Entrega

Para realizar a prova e entregar seu código, siga rigorosamente estes passos:

1. **Fork do Repositório**:
   - Clique no botão **Fork** no canto superior direito desta página para criar uma cópia do projeto na sua conta pessoal do GitHub.
   
2. **Desenvolvimento**:
   - Clone o seu fork localmente: `git clone https://github.com`
   - Realize o desenvolvimento dentro da pasta do projeto.
   - Faça commits organizados descrevendo o que foi implementado.

3. **Publicação e Envio**:
   - Dê um `push` das suas alterações para o seu repositório no GitHub.
   - Envie o link do seu repositório para o avaliador (ou conforme as instruções passadas no momento da prova).

---

## 🎨 Critérios de Avaliação

- **Fidelidade ao Design**: Proximidade com o modelo original.
- **Responsividade**: A interface deve se adaptar bem a diferentes tamanhos de tela.
- **Qualidade do Código**: Organização, nomes de classes (BEM é um diferencial) e indentação.
- **Semântica HTML**: Uso correto das tags para acessibilidade e SEO.
- **Lógica de JS**: Código limpo e eficiente para as interações.

**Boa sorte! Mostre do que você é capaz. 🔥**
